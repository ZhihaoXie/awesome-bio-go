## General Bioinformatics Libraries

- [biogo/biogo](https://github.com/biogo/biogo) - A bioinformatics library for the Go programming language. ![stars](https://img.shields.io/github/stars/biogo/biogo?style=social)

- [biogo/hts](https://github.com/biogo/hts) - SAM and BAM handling for the Go language. ![stars](https://img.shields.io/github/stars/biogo/hts?style=social)

- [shenwei356/bio](https://github.com/shenwei356/bio) - A lightweight and high-performance bioinformatics package in Golang. ![stars](https://img.shields.io/github/stars/shenwei356/bio?style=social)

- [pbenner/gonetics](https://github.com/pbenner/gonetics) — Go/Golang Bioinformatics Library. ![stars](https://img.shields.io/github/stars/pbenner/gonetics?style=social)

- [grailbio/bio](https://github.com/grailbio/bio) - Bioinformatic infrastructure libraries. ![stars](https://img.shields.io/github/stars/grailbio/bio?style=social)

- [(Poly)merase](https://github.com/bebop/poly) - Poly is a Go package for engineering organisms. ![stars](https://img.shields.io/github/stars/bebop/poly?style=social)


## Workflow and Pipeline Management

- [scipipe](https://github.com/scipipe/scipipe) - Robust, flexible and resource-efficient pipelines using Go and the commandline, compiling to a single binary, providing powerful file naming and comprehensive audit reports for every output. ![stars](https://img.shields.io/github/stars/scipipe/scipipe?style=social)

- [Reflow](https://github.com/grailbio/reflow) - A language and runtime for distributed, incremental data processing in the cloud. ![stars](https://img.shields.io/github/stars/grailbio/reflow?style=social)

- [bioflows](https://github.com/drsamibioinfo/bioflows) - Distributed Containerized Bioinformatics and Computational Workflows/Pipeline Engine built in Golang. [BioFlows’s documentation](https://bioflows2.readthedocs.io/en/latest/). ![stars](https://img.shields.io/github/stars/drsamibioinfo/bioflows?style=social)

- [Dagu](https://github.com/dagu-org/dagu) - Dagu is a lightweight workflow engine with a modern Web UI. Workflows are defined in a simple, declarative YAML format and can be executed on schedule. ![stars](https://img.shields.io/github/stars/dagu-org/dagu?style=social)

See also [Awesome Pipeline](https://github.com/pditommaso/awesome-pipeline)
